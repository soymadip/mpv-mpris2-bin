## Installation

To install `mpv-mpris2-bin`, you can use an AUR helper like `yay` or `paru`. Here are the commands for both:

- Using `yay`:

```bash
yay -S mpv-mpris2-bin
```

- Using `paru`:

```bash
paru -S mpv-mpris2-bin
```
